# WebPack Agile

Install project components - sudo yarn install

<h2 align="center">Install</h2>

Development requires an environment

<p>node - v12.18.3</p>
<p>npm - v6.14.6</p>
<p>yarn - v1.22</p>
<p></p>

Install with yarn:

```bash
sudo yarn install
```

<h2 align="center">Start</h2>

Install with yarn:

```bash
yarn start
```

<h2 align="center">Build project</h2>

Install with yarn:

```bash
yarn build
```
